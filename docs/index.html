Zeta 是一门可自举、自编译、跨平台、默认安全、可脚本化也可 AOT 编译的 AI 原生系统应用语言。

文档原则

  • 除 GitHub README.md 使用 Markdown 作为仓库首页外，其他项目文档全部使用 HTML 编写。
  • 正式文档优先使用中文，后续通过同一结构支持多语言版本。
  • 每份文档必须标明状态、更新时间、适用范围和验收标准。
  • README 作为 GitHub 入口，正式规格和决策以 HTML 文档为准。
