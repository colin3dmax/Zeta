低门槛，高上限，小内核，高性能

像 JavaScript 一样容易开始，像系统语言一样能抵达底层。

Zeta 是一门严肃的专业编程语言，面向上层应用、服务端、桌面、移动端、WASM、操作系统、嵌入式和硬件软件。核心内核保持小而强，外围能力通过标准库和第三方模块按需扩展。

当前可用能力

命令行

  • zeta ast-dump：输出 AST dump。
  • zeta check：执行 parse、name resolution 和基础 typecheck。
  • zeta repl：交互式语法终端，支持 :help、:doc、:complete。

语言前端

  • 模块、import/export、函数、struct、enum。
  • let、return、if/else、while、match。
  • Int、String、Bool 和基础算术表达式。
