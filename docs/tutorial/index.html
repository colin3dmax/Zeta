Zeta 在线教程

状态：draft

更新时间：2026-05-28

适用范围：Zeta Stage 0 用户教程，覆盖当前已经实现的语法、CLI、REPL 和检查能力。

验收标准：用户可以按章节学习当前语言能力，并能跳转到 playground 或本地命令验证示例。

课程 1：第一个模块

module demo;
import std.io;

模块声明和 import 是当前文件级结构的基础。可以在 Playground 中查看 AST。

课程 2：函数和返回值

fn main(name: String) -> Int {
  return 0;
}

当前 checker 会检查 return 表达式和函数返回类型是否一致。

课程 3：局部变量和表达式

fn calc() -> Int {
  let base: Int = 1 + 2 * 3;
  return base;
}

算术表达式目前支持 + - * /，乘除优先级高于加减。

课程 4：控制流

fn choose(flag: Bool) -> Int {
  if flag {
    return 1;
  } else {
    return 0;
  }
}

if 和 while 条件必须是 Bool。

课程 5：结构体和枚举

struct User {
  name: String,
  age: Int,
}

enum ResultTag {
  Ok,
  Err,
}

当前 Stage 0 可以解析并 dump struct / enum，后续会接入字段访问和模式匹配的类型检查。

课程 6：REPL 文档查询

cargo run -- repl
:help
:doc let
:complete st

REPL 可直接计算表达式和查询文档；完整程序可通过 zeta run 走 Stage 0 MIR interpreter 执行。
