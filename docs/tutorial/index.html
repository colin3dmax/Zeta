Tutorial

用可运行的小例子掌握 Stage 0。

教程从模块、函数、局部绑定和控制流开始，再连接到数据声明、REPL 和 Playground。每个例子都对应当前编译器已经能解析、检查或执行的能力。

课程 1：第一个模块

module demo;
import std.io;

模块声明和 import 是当前文件级结构的基础。可以在 Playground 中查看 AST。

课程 2：函数和返回值

fn main(name: String) -> Int {
  return 0;
}

当前 checker 会检查 return 表达式和函数返回类型是否一致。

课程 3：局部变量和表达式

fn calc() -> Int {
  let base: Int = 1 + 2 * 3;
  return base;
}

算术表达式目前支持 + - * /，乘除优先级高于加减。

课程 4：控制流

fn choose(flag: Bool) -> Int {
  if flag {
    return 1;
  } else {
    return 0;
  }
}

if 和 while 条件必须是 Bool；比较表达式会返回 Bool，例如 count < 3 或 answer == 42。可以用 &&、|| 和 ! 组合条件。

课程 5：结构体和枚举

struct User {
  name: String,
  age: Int,
}

enum ResultTag {
  Ok,
  Err,
}

当前 Stage 0 可以解析并 dump struct / enum，后续会接入字段访问和模式匹配的类型检查。

课程 6：REPL 文档查询

cargo run -- repl
:help
:doc let
:complete st

REPL 可直接计算表达式和查询文档；完整程序可通过 zeta run 走 Stage 0 MIR interpreter 执行。

课程 7：Float 浮点

fn main() -> Int {
  let x: Float = 3.0;
  let y: Float = 2.0;
  let r: Float = x * y - x / y;   // 4.5
  if r > 4.4 { if r < 4.6 { return 7; } }
  return 0;
}

Float 是 64 位浮点。字面量小数点后必须有数字（2.0 而非 2.）。Int 和 Float 不隐式混用；取模和位运算仅限 Int。原生后端用 LLVM double 实现。

课程 8：Tuple 元组

fn pair(a: Int, b: Int) -> (Int, Int) {
  return (a, b);
}

fn main() -> Int {
  let t = pair(10, 20);
  return t.0 + t.1;   // 按位置索引
}

元组是固定元数的异构聚合：(a, b) 构造、t.0/t.1 索引（可嵌套 t.1.0）。注意 (a) 不带逗号只是分组而非单元素元组。类型注解 (Int, String) 让元组能传参与返回。

课程 9：Closure 闭包

fn main() -> Int {
  let n: Int = 10;
  let addn = |x: Int| x + n;   // 按值捕获 n
  return addn(5);              // 15
}

Lambda |x: Int| body 是一等值，创建时按值捕获自由变量。函数类型 fn(Int) -> Int 可作参数/返回，因此可以写高阶函数和返回闭包的工厂函数。

课程 10：Generics 泛型

fn id<T>(x: T) -> T {
  return x;
}

fn main() -> Int {
  let a: Int = id(40);
  let b: Float = id(2.0);   // 同一函数，两种类型
  if b > 1.9 { return a + 2; }
  return 0;
}

泛型函数 fn id<T>(...) 的类型参数从实参推断（调用点不写 <>）。函数体内 T 是不透明类型，只能做与类型无关的操作。原生后端为每个具体实例（id$Int、id$Float）按需单态化生成专门副本。
