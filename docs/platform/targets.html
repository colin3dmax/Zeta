Zeta 跨平台与运行时架构

状态：draft

更新时间：2026-05-28

目标平台

  • Desktop / Server：macOS、Linux、Windows，x64、arm64、riscv64。
  • Mobile：iOS arm64、Android arm64，优先以 staticlib/cdylib 进入原生应用。
  • Web：wasm32-browser、wasm32-wasi、WASM Component。
  • Embedded：arm-none-eabi、riscv32-none，支持 no_std 和可选 alloc。

标准库分层

std
  -> alloc
    -> core
  • core：基础类型、Option、Result、panic 抽象，无操作系统和堆依赖。
  • alloc：堆分配、字符串、数组、集合和智能指针。
  • std：文件、网络、线程、时间、环境变量和平台集成。

Runtime 裁剪

  • runtime-none：裸机和极小嵌入式。
  • runtime-core：基础 panic、入口、内存和错误处理。
  • runtime-async：异步调度和事件循环。
  • runtime-std：文件、网络、线程、进程。
  • runtime-ai：schema、tool、agent、trace、policy。

CI Matrix

  • Host：Linux、macOS、Windows。
  • Target：native、wasm32、mobile build-only、embedded no_std。
  • Runtime：std、alloc、no_std、minimal。
  • 检查：format、lint、unit、cross-compile smoke、runtime feature matrix、WASM artifact size。
