Zeta VS Code 插件使用说明

状态：draft

更新时间：2026-05-28

适用范围：本地安装和使用 Zeta VS Code 插件，包括语法高亮、snippets、静态补全和 hover 文档。

验收标准：用户可以安装插件、重载 VS Code、打开 .zeta 文件，并看到 Zeta 语言模式、补全和悬浮文档。

一键本地安装

从仓库根目录运行：

sh editors/vscode-zeta/scripts/install-local.sh

该脚本会把插件目录链接到 VS Code 用户扩展目录：

~/.vscode/extensions/zeta.zeta-language-0.1.0

重载 VS Code

  • 打开命令面板，执行 Developer: Reload Window。
  • 或者退出并重新打开 VS Code。

打开 Zeta 文件

testdata/core_items.zeta

右下角语言模式应显示 Zeta。如果没有显示，可以点击语言模式并手动选择 Zeta。

插件能力

  • .zeta 文件关联。
  • 模块、导入、函数、结构体、枚举、控制流、字面量、注释和操作符的语法高亮。
  • module、fn、exportfn、struct、enum、let snippets。
  • Stage 0 关键词和标量类型的静态补全。
  • 关键词和类型的 hover 文档。

手动安装

mkdir -p ~/.vscode/extensions
ln -sfn "$(pwd)/editors/vscode-zeta" ~/.vscode/extensions/zeta.zeta-language-0.1.0

验证插件

code --list-extensions | grep zeta
python3 tools/check-vscode-extension.py

当前限制

  • 当前插件不是 LSP，暂不提供基于项目语义的诊断、跳转定义或重命名。
  • 补全和 hover 文档是静态规则，后续会接入 Zeta 编译器前端。
  • 修改插件源码后，需要重载 VS Code 才能看到更新。
