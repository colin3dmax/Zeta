安装

下载后解压，把 bin/zeta 复制到 PATH 中的目录。完整步骤见 本地安装。

tar -xzf zeta-0.1.0-macos-aarch64.tar.gz
cp zeta-0.1.0-macos-aarch64/bin/zeta ~/.local/bin/zeta
zeta repl
